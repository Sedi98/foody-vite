# 🍽️ Foody

**Foody** is a comprehensive food ordering app where users can browse through a list of restaurants, view restaurant details, explore available products, and manage their orders easily. It also includes a profile section where users can update their profile, change settings, and manage their orders. The app comes with a robust **Admin Dashboard** for managing products, categories, orders, and offers.

## 🌟 Features

- 🏪 View list of restaurants and their details
- 🍔 Add products to the basket and manage the basket
- 🛒 Increase, decrease, or remove products from the basket
- 👤 Create an account to manage your basket and orders
- 📸 Update profile photo and settings
- 📦 Track orders via the profile section
- 🛠️ **Admin Dashboard** for managing:
  - Products
  - Categories
  - Orders
  - Offers

---

## 🛠️ Technologies Used (Frontend)

- ⚛️ **React** with **Vite** and **Typescript** for a blazing-fast development experience
- 🎨 **Tailwind CSS** for styling
- 🌐 **i18n** for multi-language support
- 🗂️ **Context API** for state management
- 🌐 **Axios** for API requests
- 🔔 **Toastify** for beautiful notifications
- 🚀 **Vite + SWC** for optimized performance
- 🧠 **React Helmet** for dynamic HTML meta tags
- 🛠️ Plus more...

## 🔧 Technologies Used (Backend)

## 🇦🇿 Foody (Azərbaycanca)

**Foody**, istifadəçilərin restoranların siyahısını görməsi, restoran təfərrüatlarını görməsi, mövcud məhsullarla tanış olması və sifarişlərini asanlıqla idarə etməsi üçün hazırlanmış tam funksiyalı qida sifarişi proqramıdır. Proqramda istifadəçilərin profillərini yeniləyə biləcəyi, parametrləri dəyişə biləcəyi və sifarişlərini idarə edə biləcəyi profil bölməsi də var. Eyni zamanda, məhsulların, kateqoriyaların, sifarişlərin və təkliflərin idarə edilməsi üçün güclü bir **Admin Panel** mövcuddur.

## 🌟 Xüsusiyyətlər

- 🏪 Restoranların siyahısını və təfərrüatlarını görün
- 🍔 Məhsulları səbətə əlavə edin və səbəti idarə edin
- 🛒 Məhsulları artırın, azaldın və ya səbətdən çıxarın
- 👤 Səbətinizi və sifarişlərinizi idarə etmək üçün hesab yaradın
- 📸 Profil şəklini və parametrləri yeniləyin
- 📦 Profil bölməsindən sifarişləri izləyin
